# Keigo-AI 产品需求文档（PRD）

## 1. 项目概述

**Keigo-AI** 是一款面向日本商务场景的 AI 文书助手，核心目标是帮助日本用户高效撰写商务邮件、优化敬语表达、自动格式化文书，并提供智能模板库支持。产品以 Web 端形式提供，突出“简单、专业、可信赖”的特点。

---

## 2. 产品目标

1. 降低日本职场人员撰写商务邮件的时间成本。
2. 减少敬语错误，提高文书专业度。
3. 为中小企业提供高性价比的 AI 文书自动化工具。
4. 提供比通用 AI 更本地化、更符合日本商务文化的文书输出。

---

## 3. 目标用户

### 3.1 主要用户

* 日本职场人士（商务人士、白领）
* 中小企业员工
* 客服、销售、企划部门
* 对敬语不熟悉的新入职员工

### 3.2 次要用户

* 留学生、外国人在日本工作人士
* 接触大量邮件往来的自由职业者

---

## 4. 使用场景

* 撰写客户沟通邮件（报价、致歉、催促、问询）
* 生成会议纪要、报告总结等正式文书
* 优化内部沟通邮件的敬语表达
* 生成多版本邮件以供选择

---

## 5. 产品核心功能

### 5.1 输入意图 → 自动生成完整商务邮件

**说明**：用户输入需求或核心信息，系统生成格式化的商务邮件。

* 输入：句子、要点、关键词
* 输出包含：件名、挨拶、本文、結び、署名
* 输出三种语气版本：

  * 简洁版
  * 标准商务版
  * 高度敬语版
* 提供复制按钮

### 5.2 敬语校正（Keigo Correction）

用户粘贴已有邮件，系统自动：

* 修正不自然或错误的敬语
* 统一文风为商务标准
* 自然化表达
* 提供修改差异（Diff 视图，可选）

### 5.3 商务文书模板库

提供高质量日文商务模板，按分类展示：

* 道歉类
* 提案类
* 询问类
* 跟进类
* 内部报告
* 会议纪要
* 催促・提醒
* 招聘相关

每个模板可：

* 一键加载
* 自动让用户补充信息
* 自动生成三版本邮件

### 5.4 自动生成件名（主题）

根据邮件内容智能生成：

* 标准商务件名
* 简洁件名
* 正式件名

### 5.5 多版本对比

用户可查看：

* 简洁版
* 标准版
* 更礼貌版本

## 8. UI 和品牌风格要求

* 简洁、专业、柔和。
* 以白色、浅灰、淡蓝为主（日本用户偏好干净风格）。
* 采用清晰的日语文案。
* 强调信赖感（信頼・丁寧・正確）。

---

## 9. 技术方案

### 9.1 前端技术栈

* Next.js / React
* Tailwind CSS
* Axios

### 9.2 后端技术栈

* Next.js API Routes
* JWT 权限机制
* Rate Limit（防滥用）
* Redis（缓存，对话上下文管理）

### 9.3 模型

* OpenAI GPT-4o / GPT-4.1 / GPT-o1-mini
* Prompt 模板化管理

### 9.4 数据存储

* MongoDB / PostgreSQL
* 用户信息
* 模板库
* 生成记录（可选）

## 13. 风险与对策

**风险**：用户担心 AI 输出敬语不够自然。
**对策**：

* 加入“多版本输出”
* 专门强化日语 prompt
* 用户反馈系统（改善输出质量）

---

## 14. 总结

Keigo-AI 将成为日本市场上最实用的 AI 商务文书助手。通过聚焦敬语、邮件格式化和模板库功能，可快速吸引日本职场用户，具有明确市场需求和可持续商业模式。
